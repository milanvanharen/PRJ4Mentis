Usability Heuristics

Visibility of system status

On the top of the page, the user sees a scrolling bar that fills up as the user scrolls down on the page. This keeps the user informed about what is going on on the page.

Match between system and real world

We do not like reading complicated words that we do not understand as for we did not apply any in our own website. Our site is written in a normal-people-language that everyone can understand.

Error prevention and helping users recognize, diagnose, and recover from errors

Errors are not fun. For nobody. That is why both our site's contact form and sign up/log in pages have built-in error prevention. An example of this is trying to fill in an email adress without using a @. The site will notice this and the user will get a warning message saying that the email adress might be invalid.

Aesthetic and minimalistic design

Our website uses a minimalistic design by only using a few colors. This fact combined with the prominently displayed pictures makes the site aesthetically pleasing and does not distract the users' attention from what is important.
